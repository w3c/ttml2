About the TTML2 Recommendation

The TTML2 Recommendation was produced by the W3C Timed Text (TT) Working Group as part of the W3C Video in the Web Activity.

This document lists the known errata to the TTML2 Recommendation. Each entry has the following information:

  • The section referenced by the erratum.
  • A description of the problem and correction if applicable.
  • The date it was added to the errata page.

Please send general comments about this document to the public mailing list public-tt@w3.org (subscribe, archive).
